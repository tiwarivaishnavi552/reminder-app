# reminder-app 

A personal productivity dashboard — daily schedule, task/goal tracking, and
browser notifications for study/work blocks. Runs entirely client-side
(data stored in `localStorage`), built with Next.js, TypeScript, and Tailwind.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Allow notifications when
prompted to get reminders for your schedule blocks.

## Features (Phase 1)

- Daily schedule with live "current task" tracking and progress bar
- Goal checklist and study-hours tally
- Streak counter
- Browser notifications when a scheduled block starts
